check() → {void}

Checks the current state of the game board for a winner or a draw.

This function examines each row, column, and diagonal to determine if a player has won. If a winner is found, the 'access' variable is set to 0 and a message is logged to the console. The horizontal, vertical, and diagonal checks are performed separately.
